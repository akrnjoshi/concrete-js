(static) $.fn.getTokenSelectCallbacks() → {jQuery.Callbacks}

Returns the jQuery.Callbacks object for "token select" callback functions. If the jQuery.Callbacks object does not already exist, it will be created. For details about jQuery.Callbacks, see: http://api.jquery.com/category/callbacks-object/ The "token select" callback functions will not be called unless the user has also called enableTokenSelectCallbacks(). Once enableTokenSelectCallbacks() has been called, when the user selects a set of displayed tokens, all registered "token select" callback functions will be called and passed a list of TokenRefSequences containing the selected tokens.
